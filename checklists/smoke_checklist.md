# Smoke Checklist (Web App)

## Build & Access
- [ ] App loads without critical errors
- [ ] Main pages open (Home, Login, Profile)
- [ ] No broken navigation links on main menu

## Authentication
- [ ] User can log in with valid credentials
- [ ] Invalid login shows error message
- [ ] User can log out successfully

## Core Flows (Example)
- [ ] User can open a main feature page
- [ ] Primary action works (create / submit / save)
- [ ] Data is displayed correctly after action

## UI/UX
- [ ] Buttons clickable and visible
- [ ] No major layout breaks on common resolution
