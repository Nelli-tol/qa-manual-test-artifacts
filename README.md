# Manual QA Test Artifacts

This repository contains examples of manual QA documentation used in real testing workflows.

## Contents
- `test-cases/` – sample test cases (web)
- `checklists/` – smoke/regression checklists
- `bug-reports/` – bug report template + sample bug
- `test-plan/` – sample test plan document

## Notes
Artifacts are written in a Jira-style format and can be adapted for different projects.
